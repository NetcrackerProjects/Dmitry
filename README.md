# Менеджер задач

Используемые технологии 
* Java (1.8.0_192)
* Maven (3.6.0)

## Структура прокета
* [api]
* [command]
* [entity]
* [error]
* [repository]
* [controller]
* [service]
* App - основной класс 

## Сборка проекта
    mvn clean install

## Запуск
    java -jar ./task-manager-1.0-SNAPSHOT.jar
